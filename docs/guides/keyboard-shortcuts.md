# Keyboard Shortcuts

All shortcuts use ⌘ (Cmd) on macOS. Press `⌘?` to open the shortcuts dialog inside the app.

## Global

| Shortcut | Action |
|---|---|
| ⌘K | Search workspace |
| ⌘R | Smart Scan |
| ⌘Z | Undo last action |
| ⌘⇧S | Toggle sidebar |
| ⌘⇧M | Toggle Safe Mode |
| ⌘, | Open Settings |
| ⌘? | Show keyboard shortcuts |
