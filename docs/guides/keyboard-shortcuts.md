# Keyboard Shortcuts

All shortcuts use ⌘ (Cmd) on macOS. Press `⌘?` to open the shortcuts dialog inside the app.

## Global

| Shortcut | Action |
|---|---|
| ⌘K | Search workspace |
| ⌘R | Smart Scan |
| ⌘Z | Undo last action |
| ⌘E | Export as JSON |
| ⌘⇧E | Export as Markdown |
| ⌘⇧S | Toggle sidebar |
| ⌘⇧D | Toggle dark mode |
| ⌘⇧M | Toggle Safe Mode |
| ⌘, | Open Settings |
| ? | Show keyboard shortcuts |

## Zoom

| Shortcut | Action |
|---|---|
| ⌘= | Zoom in |
| ⌘- | Zoom out |
| ⌘0 | Actual size |
